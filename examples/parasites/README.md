# Parallax Showcase — `examples/parasites`

Three cinematic scroll-driven landing pages. Each one is a single locked viewport — the page itself never scrolls. Your scroll gesture *scrubs the frames* of a short video in place while five Anton/Caveat text scenes crossfade in and out.

> This is the **live, working output** of the [`parallax-landing-page`](../../skills/parallax-landing-page/SKILL.md) skill that lives in this same repo. Treat it as the ground-truth reference if the skill's templates are ever ambiguous.

Enter via [`index.html`](index.html) — the hub.

## How it works

- Every video frame is extracted at native 1280×720 with `ffmpeg -q:v 2` into `<slug>/frame-NNN.jpg`.
- `parallax.js` locks the body (`body.scrub-page`), intercepts wheel/touch/keyboard, and converts deltas into a virtual `0..1` progress.
- A RAF loop lerps the current frame toward the target (factor `0.22`) for buttery scrubbing.
- Five scene overlays with `data-start` / `data-end` progress windows fade in and out with a trapezoidal curve and a 26px upward translate.
- The final scene contains the CTAs — there is no separate `.end` section beneath the hero.

## Run it locally

```bash
cd parasites
python -m http.server 8000
# open http://localhost:8000/
```

`file://` is not recommended — Chromium can taint the canvas with local image origins, and the frames may never paint.

## Typography contract

- **Anton** (Google) — every uppercase headline, kicker, nav, CTA
- **Caveat** (Google) — handwritten accent lines in gold / amber / cream
- **Inter** — quiet body lead text only
- Off-white `#f5f1ea`, never pure `#fff`
